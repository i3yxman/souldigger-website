# 公开匿名案例模板（官网可直接发布）

## 使用说明
- 先选一个最接近的行业模板，再替换方括号内容。
- 所有客户识别信息必须脱敏：公司名、门店名、人员名、精确地址、原始截图。
- 指标必须是“可解释、可复核”的真实口径，避免绝对化表达。

## 模板A：连锁零售（神秘顾客 + 明访巡检）

### 标题
某全国连锁零售品牌通过 Vantara 建立统一巡检与整改闭环体系

### 背景（匿名）
- 行业：零售
- 规模：[X]个城市 / [X]家门店 / [X]位区域管理者
- 项目类型：神秘顾客与明访巡检并行

### 挑战
- 跨区域执行标准不统一，评分不可比
- 现场证据分散，复核争议多
- 整改任务无统一跟进机制

### 方案
- 建立统一任务模板与评分口径
- 移动端采集文本、图片、定位、时间证据
- 配置“提交-复核-驳回-整改-复查”流程
- 建立管理看板与周度异常复盘

### 上线过程
- 第1周：流程梳理与字段标准化
- 第2周：试点区域上线与审核规则校准
- 第3周：问题分类优化与异常闭环固化
- 第4周：扩展至全量区域

- 巡检准时完成率：`[A%] -> [B%]`
- 审核平均处理时长：`[A小时] -> [B小时]`
- 异常闭环周期：`[A天] -> [B天]`

### 公开评价（可选）

- 规模：[X]个城市 / [X]家经销门店
